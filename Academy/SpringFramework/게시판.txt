1. 사용자 인증을 기반으로 한 Q&A 게시판을 개발한다. 
2. 회원은 질문을 등록하고, 다른 사용자는 댓글(답변)을 남길 수 있다. 
3. 질문/답변 모두 본인만 수정 및 삭제 가능하다.
4. 인증되지 않은 사용자는 글 읽기만 가능하다.

회원가입 / 로그인   Spring Security 기반 로그인
질문 목록 조회   전체 질문 목록을 최신순으로 조회
질문 등록      로그인한 사용자가 질문 작성
질문 상세 조회   질문 내용 + 답변 목록 표시
질문 수정/삭제   작성자만 수정/삭제 가능
답변 등록      로그인한 사용자가 특정 질문에 댓글 형식으로 답변 등록
답변 수정/삭제   작성자만 수정/삭제 가능

질문 게시판

URL         설명
GET   /api/questions   질문 목록 조회 (페이징 지원 가능)
GET   /api/questions/{id}   특정 질문 + 답변 목록 조회
GET   /api/questions   질문 작성 폼 //앞의 질문 목록 조회랑 겹치는데?  /api/questionsForm으로 작성함.
POST   /api/questions   질문 작성(insert)
GET    /api/questions/update/{no}   질문 수정 폼
PUT   /api/questions/update/{no}   질문 수정 (작성자만 가능)
DELETE   /api/questions/{id}   질문 삭제 (작성자만 가능) - 답변까지 다 삭제

답변 (댓글) 게시판 

URL   설명
POST   /api/questions/{questionId}/answers   특정 질문에 답변 작성  //댓글쓰기 폼으로 이동 추가(get)
PUT   /api/answers/{id}   답변 수정 (작성자만 가능)
DELETE   /api/answers/{id}   답변 삭제 (작성자만 가능)

게시판을 만들거야.
회원 가입한 사람들만 글을 읽고 쓸 수 있고,
가입에는 ID, 비밀번호, 닉네임, 이메일이 필요해.
글은 본인만 수정, 삭제가 가능해. 이건 댓글도 마찬가지야.
단 게시글과 답변을 처음 생성 시, 비밀번호 유무를 체크해서 비밀번호를 걸 수 있는데, 이 비밀번호를 알면 누구라도 삭제, 수정이 가능해.
비밀번호를 걸겠다고 체크박스에 체크하면 별도의 공간이 떠서 거기에 글을써서 비밀번호 저장이 되. 체크박스 해제하면 공간은 사라지고.
등급은 일반, 우수, 관리자 가 있는데,
관리자는 회원의 정보 연람 및 수정이 가능하고, 모든글의 수정, 삭제도 가능해.
게시판은 검색 기능(닉네임과 글의 제목으로)이 있고, 답변과 게시글에 좋아요, 싫어요. 기능이 있어.
검색은 사용자의 선택에 따라 시간순, 닉네임, 좋아요, 싫어요. 순으로 정렬이 가능해
처음 index화면에 로그인,회원가입이 있고, 로그인하면 접근가능한 기능들에 대한 링크가 index에 출력.

비로그인->로그인, 회원가입,QnA 게시판(읽기만 가능,비밀번호를 알면 수정,삭제가능)
일반 -> 회원정보 수정, 탈퇴, QnA 게시판(글쓰기,수정,읽기,삭제 및 기존글의 답변,답변 수정, 답변 삭제 가능)
게시글 작성시 제목,내용을 작성자가 적고, 닉네임과 작성 시간은 자동으로 출력됨.
수정 된 글은 수정된 글임을 알 수 있고, 수정된 시간도 같이 나와.

우수 ->일반의 기능 + 게시글과 답변에 좋아요 / 싫어요 기능(한 아이디에 대해서 각 게시글과 각 답변 당 한번씩만 가능)
좋아요나 싫어요를 다시 누르면 (즉 2번누르면) 취소가 가능.
좋아요 누르고서 싫어요는 못누름(반대도 불가)

관리자 -> 우수 + index에 회원관리 추가(회원정보 열람 및 수정, 삭제 가능)+ 모든 게시글, 답변의 수정 삭제 가능하고, 비공개 처리도 가능해

그리고 일반은 index 화면에서 승급요청이라는 버튼이 있어서 누르면 관리자에게 승급을 요청가능.
관리자가 로그인하면 관리자 index 화면에서 몇명이 승급요청을 했는지 볼 수 있고, 앞에서 말했듯 회원정보 열람에서 등급 조정.
회원정보에는 회원의, 비밀번호, 닉네임, 이메일 뿐 아니라 작성한 글의 수와 작성한 답변의 수 열람가능. 총 몇개나 좋아요/싫어요를 받았는지까지.

승급은 관리자가 회원 정보를 보고 직접 결정.
일반, 우수 회원의 등급이 변경되면 당사자가 로그인 시 index에서 알림 출력.

관리자는 회원의 모든 정보를 수정/ 삭제가 가능

회원이 탈퇴하면 글과 답변은 남아 있어서, 비밀번호를 모르면, 관리자만 수정, 추가 가능.

게시글 목록 및 게시글에서 조회수도 확인이 가능.

게시글 목록은 10개씩, 답변은 5개씩 해서 페이징.
