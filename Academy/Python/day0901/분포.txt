| 분포                      | 특징                            | 예시                   |
| ----------------------- | ----------------------------- | -------------------- |
| **정규분포 (Normal)**       | 평균 μ, 표준편차 σ로 결정. 종 모양, 좌우 대칭 | 키, 시험 성적, 오차         |
| **t-분포 (Student's t)**  | 정규분포와 비슷하나 표본이 적을 때 꼬리가 두꺼움   | 소표본 평균 검정            |
| **카이제곱분포 (Chi-Square)** | 제곱합 분포, 0 이상만 가능              | 독립성 검정, 분산 검정        |
| **F-분포 (F)**            | 두 분산의 비율, 0 이상만 가능            | ANOVA(분산분석)          |
| **이항분포 (Binomial)**     | n번 시도, 성공 확률 p. 이산형           | 동전 n번 던지기, 시험 합격/불합격 |
| **포아송분포 (Poisson)**     | 일정 시간/공간에서 사건 발생 횟수           | 하루 교통사고 건수, 콜센터 전화 수 |
| **균등분포 (Uniform)**      | 모든 값이 동일한 확률                  | 주사위, 난수              |
| **지수분포 (Exponential)**  | 사건 발생까지 걸리는 시간                | 버스 도착 대기 시간, 부품 수명   |
| **감마분포 (Gamma)**        | 지수분포의 일반화, 대기 시간의 합           | 대기행렬, 신뢰성 분석         |
