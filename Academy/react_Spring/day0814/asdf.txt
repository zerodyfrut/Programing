1. 로그인 기능 : (관리자만 상품을 삭제가능하도록, 등록과 구매는 로그인한 사람만, 물건을 보는건 누구나 가능)

2. 재고 : 제품을 등록한 사람이 재고숫자를 늘릴 수 있음, 만약 재고가 다 떨어지면 SoldOut 상태로 되면서 
           해당 제품이 리스트에서 안보이게됨.
           등록한 사람이 로그인하면 알람이 오도록(toast나 기타 등등)
3. 계좌 충전 : 로그인한 사람은 고유의 계좌?가 있으며, 계좌에 충전된 금액만큼 제품을 구매가 가능하다.
                 잔액이 부족하면 부족하다고 알람이 뜨며 구매가 안되도록.(toast 등)
                 (장바구니에 넣는건 상관없음.)
4. 검색 : 제품 이름으로 검색. 검색어가 포함된 제품을 제품리스트로 출력
5. 상품 리뷰(text) 별점(0-5), 구매날짜(Date?)
6. 구매할때 한번에 여러개 구매가 가능하도록 수정
7. 장바구니를 전부 비우는 기능 추가
8. 장바구니에 체크박스? 등으로 선택한 제품만 구매할 수 있는 기능, 
   이때 구매여부를 묻는 알림과 구매금액이 나오고, 확인을 누르면 결제.
